# Regles Markdown

Tous les fichiers Markdown generes doivent respecter la configuration markdownlint.

**Avant de generer ou modifier un fichier Markdown, lire `.markdownlint.yaml` pour connaitre les regles actives.**

## Verification

```bash
npx markdownlint-cli2 "chemin/vers/fichier.md"
```
